# Voice Controlled Robotic Vehicle (ROS2)

#Project Overview
This project presents a voice-controlled robotic system developed using TurtleBot3 and ROS2. The system allows users to control a mobile robot using voice commands, providing a more natural and accessible interaction method, especially for individuals with limited mobility.

The system integrates offline speech recognition with real-time robot control and autonomous navigation.

---

#Features
- Voice-controlled robot movement (forward, left, right, stop)
- Offline speech recognition using Vosk
- UDP communication for real-time command transmission
- ROS2-based robot control system
- Autonomous navigation using Nav2 (partially implemented)
- SLAM mapping and localisation
- Prototype delivery mechanism (3D-printed bracket)

---

#System Architecture
Voice Input → ReSpeaker → Vosk → UDP → ROS2 → TurtleBot3

---

#Technologies Used
- ROS2 (Humble)
- TurtleBot3
- Python
- Vosk Speech Recognition (offline AI)
- UDP Socket Programming
- SLAM Toolbox
- Navigation2 (Nav2)
